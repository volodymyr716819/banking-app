# Banking App

## Project Structure

```
banking-app/
├── frontend/   # Vue 3 + Vite
├── backend/    # Spring Boot + Maven
```

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs on: [http://localhost:5173](http://localhost:5173)

### Backend

```bash
cd backend
mvn spring-boot:run
```

Runs on: [http://localhost:8080](http://localhost:8080)
